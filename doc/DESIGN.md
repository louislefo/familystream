
# Project Brief: FamilyStream

## 1. Project Overview
FamilyStream is a high-end, minimalist video streaming platform designed for a cinematic viewing experience. Inspired by modern interfaces like Apple TV+ and Netflix, it focuses on content immersion through a dark, sleek aesthetic and intuitive navigation.

## 2. Target Audience
*   Movie enthusiasts seeking a premium, distraction-free interface.
*   Users who appreciate modern UI trends like glassmorphism and deep dark modes.

## 3. Core Features
*   **Hero Banner:** A high-impact visual section on the homepage featuring the "Movie of the Moment."
*   **Content Discovery:** Categorized horizontal sliders and a dedicated "Browse" grid for easy exploration.
*   **Movie Details:** Comprehensive metadata including synopsis, cast & crew, and intelligent recommendations.
*   **Immersive Video Player:** A responsive 16:9 player with a "total dark mode" surrounding the content to eliminate distractions.
*   **Hover Interactions:** Interactive movie cards that reveal metadata (title, rating, play button) only on user engagement.

## 4. Visual Identity (UI/UX)
*   **Color Palette:**
    *   Primary Background: Noir Profond (#050505)
    *   Component Surface: Dark Gray (#1A1A1A)
    *   Accent Color: Electric Blue (Primary) or Violet
*   **Typography:** Modern Sans-serif (Inter).
*   **Effects:**
    *   **Glassmorphism:** Applied to the Top Navigation Bar for a layered, sophisticated feel.
    *   **Smooth Transitions:** Hover states and layout shifts should feel fluid.
*   **Layout:** Responsive desktop-first design with a focus on 16:9 aspect ratios for video content.

## 5. Technical Requirements
*   **Responsive Iframe:** The player must maintain a 16:9 ratio across all screen sizes.
*   **Shared Components:**
    *   `TopNavBar`: Fixed position with glassmorphism and search functionality.
    *   `Footer`: Minimalist design with essential links.
    *   `MovieCard`: Standardized component for grid and slider layouts.

## 6. Success Metrics
*   **User Engagement:** Time spent browsing vs. watching.
*   **Visual Consistency:** Seamless transition between discovery and playback phases.
